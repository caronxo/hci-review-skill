## 🔍 How to use the skill pack

Use the skill pack in a simple review loop:

1. Pick the screen or flow you want to check.
2. Write the goal for the user.
3. Move through each step as a real user would.
4. Look for places where the user may get stuck.
5. Note unclear labels, hidden actions, and extra steps.
6. Check if each screen supports the user goal.
7. Save the issues in plain language.

For a stronger review, ask these questions:

- Can a new user understand this screen?
- Does the next step feel clear?
- Can the user recover from a mistake?
- Is the text easy to read?
- Does the design match the task?

## 📚 Common review tasks

### 👀 Cognitive walkthrough
Use this when you want to test how a new user moves through a flow. Focus on what the user tries to do, what they expect, and what the system shows next.

### 🧩 Heuristic evaluation
Use this when you want to judge the interface against common UX rules. Check for clear feedback, simple wording, and consistent controls.

### 📝 Research planning
Use this when you need to plan user interviews, tests, or review sessions. Define the goal, the users, and the questions you want to answer.

### 🎯 Interface critique
Use this when you want to give direct feedback on layout, text, and behavior. Keep notes short and clear.

### 🧪 Usability review
Use this when you want to find friction in a screen or flow. Look for confusion, delays, and extra work.
